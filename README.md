# Streamlit UI Craft

A [Agent Skill](https://cursor.com/docs/context/skills) for building distinctive, production-grade UI in Streamlit apps — without fighting Streamlit's default look.

## Install

Copy this folder into a Cursor skills directory:

- **Personal:** `~/.cursor/skills/streamlit-ui-craft/`
- **Project:** `.cursor/skills/streamlit-ui-craft/`

The folder must include `SKILL.md` at its root. Do not install under `~/.cursor/skills-cursor/` (reserved for Cursor built-ins).

## Contents

```
streamlit-ui-craft/
├── SKILL.md
├── references/
│   ├── streamlit-hatches.md
│   └── aesthetic-recipes.md
└── README.md
```

## Usage

Cursor applies this skill when you build, style, theme, or polish a Streamlit app — including requests to make a dashboard look less generic, configure `.streamlit/config.toml`, inject CSS, or use `components.html` / custom components.

See `SKILL.md` and `references/` for design methodology, escape-hatch patterns, and aesthetic recipes.
